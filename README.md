# Quick 3D MMORPG
Absolutely 100% unsupported, and full of infuriating bugs.

## Prerequisites
- Node.js installed
- npm installed

## Installation
```sh
cd server
npm install
```

## Usage
1. Start the server
   ```sh
   cd server
   npm start
   ```
2. Open [http://localhost:3000](http://localhost:3000) in your browser
